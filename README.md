# Security Research

Public notes from my smart contract security practice.

I'm Tammy (@TammyBuilds, github.com/justtammy2), transitioning from
Solidity development into security research. This repo is where I
write up what I learn — from Ethernaut challenges, CodeHawks First
Flight contests, and audit reports I study on Solodit.

The goal is to build pattern recognition through public work.
Everything here is written for my future self first — clear enough
that I could re-read it in six months and rebuild the reasoning.

## Structure

- `ethernaut/` — Solutions and lessons from Ethernaut CTF levels
- `first-flights/` — Findings from CodeHawks First Flight contests
- `solodit-notes/` — Pattern notes from audit reports

## Format

Each writeup covers: the vulnerability, why it works at the code
level, the exploit, and the mitigation. Ethernaut writeups also
include what tripped me up — the personal debugging log matters
more than the solution.
